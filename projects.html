    Annotation App

    Used python, langchain, and openai api to create an application that loads all my class notes and allows me to query information, returning the page and quotations within my notes that are the most related to the query. The application also has a second function which allows it to annotate both video and text content in the same format as the uploaded class notes. This makes it so that lectures, YouTube videos, and large pieces of text can be annotated similar to the way I write my notes!

    Game Rec App

    Used React, Javascript, and Typescript to fetch video game data to create a game recommendation application. It searches through popular games by genre, title, and platform (PS5, Xbox, PC, Nintendo Switch). Allowing users to search for their favorite genres of games and check if they are available to buy on their platform. The application fetches the game card and displays information about the game like price of game and average user rating.

    Camping App

    Used Swift UI to create a camping app that uses a rec.gov API to search for campsites across the United States and display them based on their coordinates. When campsites are clicked, activities, nearby hospitals, and weather are displayed. The utilities of the campsite and whether it is pet-friendly are also displayed.
